# think_about_it
## --- 2019.03.22 ---

中午吃午饭的时候和一起实习的同事ZS闲聊，说起当前自然语言处理技术的瓶颈，陈丹琪的爆款博士论文总结了当前机器阅读的发展状况，
提到 现在的自然语言处理模型 无法处理 类似于 ‘法国队淘汰了阿根廷队’，到底是谁踢赢了谁，另外，在文本上加上一句无关的干扰语句，
模型性能大幅下降。我个人觉得，现在的深度学习技术，实际上还只是一种 局部特征的提取，最后组成类似于‘词袋’ 的 **高级特征袋**，
只能处理这个特征**有**或者**没有**，对于特征整体的组织关系，相对位置，基本上是没有感知的。很多论文里说 模型的 浅层 提取 语法语义，
我个人对此持怀疑态度 现在的 模型 估计根本 没有真正的**理解语法**的能力，最基础的主被动关系都难以理解，现在模型 很可能 还是靠穷尽
**高级特征**统计（有无、强弱）来发挥作用的，如果是这样 说明现在的深度学习 在智能形式上并没有比传统的统计机器学习有本质的进步
（类似于Xgb 的决策树在很多任务上往往能取得与神经网络相当的水平）。

    这一点，在CV上也能得到佐证，人脸识别机器可以做得比人好，正是因为 现在的神经网络模型，很善于捕捉局部特征，
与人的视觉相比 差异主要在下面：

    人眼提取的是偏向于 整体结构 的整体特征：大家都有这样的经历，班里的一个女生从大额头换成 齐刘海儿，
或者从齐刘海儿 突然把头发全扎上去，大家就会有一种完全换了个人 的感觉，但其实这个人 的嘴巴鼻子眼睛 脸型 都没有变。

    基于深度学习技术的人脸识别，对于这种变化则 基本不敏感，它还是能 准确地认出这个人，因为 它对比的是单个特征：
眼睛 鼻子 嘴巴 下巴……这些都很像，所以它认为没问题，笃定这是同一个人。你ps把嘴巴给他P到额头上，它也察觉不出任何异常。 
Hinton 的胶囊网络号称能解决神经网络的这个问题，让模型能够捕捉特征的相对关系，但 这一点的效果 好像还没有什么有力证明。
